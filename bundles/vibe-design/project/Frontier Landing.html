Talk to an engineer →
07.01
Does my code leave my machine?
In local mode, no. The model runs on-device and the index stays on your disk. Cloud mode routes through our EU/US regions with customer-managed keys — but you choose per-project.
07.02
Which editors does it replace?
Frontier is a standalone editor. It imports your VS Code keybindings, settings, and extensions on first launch.
07.03
What languages are supported?
47 languages via native LSP and treesitter. First-class support for TypeScript, Rust, Go, Python, Swift, Kotlin. Others are added monthly.
07.04
Can I bring my own model?
Yes. Any OpenAI-compatible endpoint works. Enterprise plans support fine-tuned Frontier models hosted in your VPC.
07.05
How is this different from an extension?
We built the whole editor around the agent loop — not bolted on. The indexer, UI, keybindings, and diff format are designed together, which is why latency lands at 38ms.
07.06
When is Windows / Linux?
Linux is in public beta. Windows ships Q3. Sign up for the waitlist and we'll email once your platform's build lands.
